# Confirmation: The application of the canonical commutation relation is obtained using the following:
        ½r_j*r_k*O(2)_jk 
        = ½r_j*r_k*(O(2)_jk + O(2)_jk^T)/2 + ½r_j*r_k*(O(2)_jk - O(2)_jk^T)/2 
        -> data_2nd = (O(2)_jk + O(2)_jk^T)
    And we now move to the asymmetric part, using r_j*r_k-r_k*r_j = i*Ω_jk, using the fact that the labels are 
    arbitrary and Ω_jk = -Ω_kj:
        ½r_j*r_k*(O(2)_jk - O(2)_jk^T)/2 
        = r_j*r_k*(O(2)_jk - O(2)_kj)/4 
        = (r_j*r_k*O(2)_jk - r_k*r_j*O(2)_kj - i*Ω_jk*O(2)_kj)/4 
        = -i*(Ω_jk*O(2)_kj)/4 
        = -i*(Ω_jk*(O(2)_kj-O(2)_kj^T))/8
